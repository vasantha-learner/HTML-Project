Intro

Definition

CSS (Cascading Style Sheets) is a stylesheet language used to control the presentation of HTML elements on a webpage, including layout, colors, fonts, spacing, and more.

Types of CSS

  • Inline CSS: Written inside an HTML element using the style attribute.
  • Internal CSS: Written inside a <style> tag within the HTML <head> section.
  • External CSS: Written in a separate .css file and linked using <link> tag.

Implementation Ways

  • Using the style attribute for inline styling.
  • Adding <style> tags in the <head> for internal styling.
  • Linking an external CSS file using <link rel="stylesheet">.

Advantages

  • Separation of content and presentation.
  • Reusability of styles across multiple pages.
  • Improved maintainability and readability.
  • Better control over layout and design.
  • Reduces HTML code redundancy.

Disadvantages

  • Browser compatibility issues may arise.
  • Can be confusing for beginners with multiple CSS types.
  • Complex stylesheets may be difficult to maintain without proper organization.
  • Excessive use of inline styles can override external/internal CSS, making maintenance harder.

Selectors

Definition

Selectors are patterns used in CSS to select HTML elements you want to style.

Types of Selectors

  • Element Selector
  • Class Selector
  • ID Selector
  • Attribute Selector
  • Pseudo-class Selector
  • Pseudo-element Selector
  • Group Selector

Advantages

  • Precise control over which elements to style.
  • Reduces repetitive code.

Disadvantages

  • Complex selectors can be hard to read.

Box Model

Definition

Every HTML element is considered a rectangular box. The CSS box model describes the content, padding, border, and margin of these boxes.

Components

  • Content: The text or media inside the box.
  • Padding: Space between content and border.
  • Border: Surrounds padding and content.
  • Margin: Space outside the border.

Advantages

  • Helps control spacing and layout precisely.

Disadvantages

  • Misunderstanding padding and margin can break layout.

Color Theory

Definition

Color theory in CSS deals with using colors effectively in web design for aesthetics and accessibility.

Implementation Ways

  • Named Colors (e.g., red, blue)
  • HEX Codes (e.g., #ff0000)
  • RGB/RGBA (e.g., rgb(255,0,0))
  • HSL/HSLA (e.g., hsl(0, 100%, 50%))

Advantages

  • Enhances visual appeal.
  • Improves user experience.

Disadvantages

  • Poor color choices can reduce readability or accessibility.

Typography

Definition

Typography in CSS refers to styling text including font, size, spacing, alignment, and decoration.

Implementation Ways

  • Font family, size, and weight.
  • Line height and letter spacing.
  • Text alignment, decoration, and transform.

Advantages

  • Improves readability and visual hierarchy.
  • Enhances user engagement.

Disadvantages

  • Poor typography can make content difficult to read.
  • Too many font styles can clutter the page.
